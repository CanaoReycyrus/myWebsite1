Welcome to website

This is my first website for publishing about github respository

    
1. Create a folder on the desktop and name anything you want.
2. Create your html file inside the folder
3. Prepare to upload your files in the github repository
	a. Type "git init"
	b. "git add ."	or all files type "git add ."
	
4. Create your github repository. Make sure you have github account.
5. Name your repository anything you want.
	a. Make sure the visibility is in public not in private
	b. Do not do anything to the settings.
	c. Click the green button "Create Repository"
6. Do not close the message site of the newly created repository. 
   You will be using the commands provided by the github repository.
7. Make sure the files in your folder are ready to be served, by typing "git status"
8. Copy the first three lines in your repository
	echo "# myWebsite" >> README.md
	git init
	git add README.md
9. Copy the last 4 lines in your repository
	git commit -m "first commit"
	git branch -M main
	git remote add origin https://github.com/bennycrispio/myWebsite.git
	git push -u origin main

10. Go back to your repository account and refresh the page.
GITHUB Pages Settings
11. Go to settings -> go to pages
12. Select the branch	-> main or master -> click the save button

Optional
13. Go to the " Code" area to check the process.
14. Wait until the process are already build by checking the color "green".
15. Upon checking if it is successfully build then -> go to settings -> page
16. You will be seeing the published website at the top portion.
